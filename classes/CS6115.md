## General Information

A class about proving the correctness of software using the
programming language/interactive theorem prover Coq.

## Prerequisites

CS 6110

## Topics Covered

- Programming in Coq
- Developing proofs
  - Induction
  - Modeling systems

## Workload

Medium, problem set every 1–2 weeks.
